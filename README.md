# 🚚 Logística de Alta Performance: Análise de SLA e Eficiência de Entregas

Este projeto foca na camada de engenharia e análise de dados para o setor logístico. Utilizando o dataset real do E-commerce brasileiro (Olist), desenvolvi queries complexas em SQL para transformar dados brutos em métricas de decisão de negócio.

## 🎯 Objetivo
Analisar os gargalos de entrega entre diferentes estados do Brasil, medir o cumprimento do SLA (Service Level Agreement) e identificar falhas na cadeia de suprimentos (distância entre estimativa e entrega real).

## 🛠️ Tecnologias de Destaque
* **SQL Avançado:** Uso intenso de CTEs (Common Table Expressions), Window Functions, CASE statements e JOINS complexos.
* **Modelagem de Dados:** Relacionamento entre múltiplas tabelas de pedidos, clientes e geolocalização.
* **Power BI:** Visualização da performance logística por região.
## 📁 Fonte dos Dados
Os dados utilizados neste projeto foram extraídos do [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce), um conjunto de dados real de 100 mil pedidos realizados entre 2016 e 2018 no Brasil.
